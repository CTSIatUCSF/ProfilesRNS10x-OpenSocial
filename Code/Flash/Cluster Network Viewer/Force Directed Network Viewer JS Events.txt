Events issued to the javascript callback function by the Force-Directed ("Cluster") Network Browser flash control
--------------------------------------------------------------------------------------------------------

 NODE_CLICK			- user clicked the node
 NODE_SHIFT_CLICK		- user clicked the node while holding SHIFT
 NODE_SHIFT_ALT_CLICK		- user clicked the node while holding SHIFT + ALT
 NODE_SHIFT_ALT_CTRL_CLICK	- user clicked the node while holding SHIFT + ALT + CTRL
 NODE_ALT_CLICK			- user clicked the node while holding ALT
 NODE_ALT_CTRL_CLICK		- user clicked the node while holding ALT + CTRL
 NODE_CTRL_CLICK		- user clicked the node while holding CTRL
 NODE_SHIFT_CTRL_CLICK		- user clicked the node while holding SHIFT + CTRL

 NODE_???_DBLCLICK		- user double clicked the node (this functionality may not be active in the current codebase)


 NODE_SELECTED			- user shift clicked the node when the control is configured to allow selections
 NODE_DESELECTED		- user shift clicked the node when the control is configured to allow selections
 NODE_IN			- mouse cursor has entered the node
 NODE_OUT			- mouse cursor has exited the node

 EDGE_IN			- mouse cursor has entered the edge
 EDGE_OUT			- mouse cursor has exited the edge
 EDGE_CLICK			- user clicked the edge
 EDGE_SHIFT_CLICK		- user shift clicked the edge when the control is configured to disallow selections
 EDGE_SELECTED			- user shift clicked the edge when the control is configured to allow selections
 EDGE_DESELECTED		- user shift clicked the edge when the control is configured to allow selections

 DUMP_NODES			- results of the dumpNodes() command
 NETWORK_LOADED			- notify JS class that the flash successfully loaded the network graph
